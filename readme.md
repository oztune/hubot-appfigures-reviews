# hubot-appfigures-reviews

appFigures reviews reader for Hubot

## Installation

In your hubot repository, run:

`npm install hubot-appfigures-reviews --save`

Then add **hubot-appfigures-reviews** to your `external-scripts.json`:

```json
["hubot-appfigures-reviews"]
```
